# 3DPROJECT
🌐 3D Interactive Website
📌 Project Overview

This project is a 3D Interactive Web Experience built using HTML, CSS, and JavaScript with smooth animations powered by the GSAP library.

The website focuses on creating modern UI animations, scroll-based effects, and immersive 3D-like transitions to enhance user interaction and visual experience.

🚀 Features

✅ Smooth 3D animations
✅ Scroll-based transitions
✅ Modern interactive UI
✅ GSAP-powered animations
✅ Responsive design
✅ High-performance animation rendering
✅ Creative landing page experience

HTML5

CSS3

JavaScript (ES6)

Animation Library
